## 程式碼範例

這是本站使用的一個簡單 TypeScript 工具函式：

```typescript
export function cn(...inputs: (string | undefined | null | false)[]): string {
  return inputs.filter(Boolean).join(" ");
}
```

這個工具函式可以合併 CSS 類名，同時過濾掉假值——對於條件樣式很有用。
